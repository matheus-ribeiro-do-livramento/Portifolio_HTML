Dicionário de Tags HTML

Abaixo estão as tags utilizadas na construção deste site e suas respectivas funções:


<html>
É a tag raiz. Define que todo o documento é uma página HTML.

<head>
Contém informações e configurações da página, como o título e os links de CSS. Não é visível para o usuário.

<body>
Define o corpo da página. Tudo o que é visualizado no navegador fica dentro dessa tag.

<a>
Cria um link (hiperlink) que permite navegar para outras páginas ou sites externos.

<img>
Serve para carregar e exibir uma imagem na tela.

<meta>
Fornece metadados sobre o documento, como configurações de caracteres e de visualização no navegador.

<title>
Define o título da página que é exibido na aba do navegador.

<link>
Importa arquivos externos para a página, como os arquivos de estilo CSS e as bibliotecas de ícones.

<header>
Define o cabeçalho da página ou de uma seção, agrupando a logo e o menu principal.

<nav>
Define uma seção de navegação, agrupando os links para outras páginas do site.

<main>
Representa o conteúdo principal e exclusivo do documento.

<aside>
Representa um conteúdo lateral e complementar ao conteúdo principal, como a sua barra lateral (sidebar).

<section>
Define uma seção genérica no documento, agrupando um tema específico de conteúdos (como a seção de disciplinas).

<footer>
Define o rodapé da página, que geralmente contém informações de direitos autorais e contatos.

<div>
É um contêiner genérico e "sem significado" estrutural, muito usado para agrupar blocos e aplicar estilos CSS.

<h1> e <h2>
São tags de título (headings). O h1 é o título principal da página e o h2 é um subtítulo de segundo nível.

<p>
Define um parágrafo de texto.

<strong>
Destaca um texto dando-lhe forte importância, deixando-o com visual de negrito.

<br>
Força uma quebra de linha simples no texto.

<i>
Originalmente para texto em itálico, hoje é a convenção padrão para adicionar ícones na tela (como os do FontAwesome).

<ul> e <li>
O ul cria uma lista não ordenada (com marcadores), e o li define cada item dentro dessa lista.

<dl>, <dt> e <dd>
Trabalham juntas para criar uma lista de definições. O dl é a lista, dt é o termo e dd é a descrição.

<address>
Fornece informações de contato para o autor ou proprietário do site no rodapé.
